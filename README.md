# 🗼 SPECTRUM KONG

A retro 8-bit arcade game inspired by Donkey Kong, built for the WISP 
(Wireless Internet Service Provider) community worldwide.

Play as a WISP technician climbing telecom towers while dodging competitor 
barrels, weather storms, and regulatory fines. Collect spectrum tokens, 
install antennas, and defeat the Interference King!

## Features
- 🎮 4 levels + boss fight with unique mechanics
- 🌐 3 languages: English, Español, Português
- 🔊 Synthesized 8-bit soundtrack (Web Audio API, no files)
- 📡 Radio wave attack system for the boss fight
- 🏆 High score table with localStorage persistence
- 📦 Single HTML file — no dependencies, no build step
- 🖥️ Pure Canvas API pixel art — no images needed

## How to Play
Open `spectrum_kong.html` in any modern browser. That's it.

- Arrow keys / WASD — Move & Climb
- SPACE — Jump
- E — Install antennas & backhaul
- X — Shoot radio waves (boss fight)

## Levels
1. 🏗️ Rural Water Tower — dodge competitor barrels
2. 🌾 Grain Elevator — weather barrels climb towers!
3. 🏙️ Rooftop POP — all hazard types + spectrum ghosts
4. ⚡ Boss: Interference Storm — collect licensed spectrum, defeat the boss

## Easter Eggs
Type "WISPA" on the title screen for rainbow mode 🌈
Type "MIMOSA" for 30 seconds of invincibility 💪
